# Bitcoin IRA Test Suite - Typescript

## Install Test Suite Dependencies

```bash
npm install
```

## Run Tests

### Specific Test Case
```bash
npx mocha -r ts-node/register 'tests/**/*.ts' --grep 'Login'
```

### Run All Test Cases
```bash
npm test
```
